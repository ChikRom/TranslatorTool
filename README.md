# Вариант №4: Инструмент командной строки для учебного конфигурационного языка

В рамках данного проекта был реализован **инструмент командной строки**, который преобразует текст на учебном конфигурационном языке в формат TOML.  
Программа считывает входной файл, анализирует его, выполняет вычисления константных выражений и формирует выходной файл.

---

## Основной функционал

- **Обработка комментариев:**  
  Однострочные комментарии, начинающиеся с символа `*`, игнорируются при обработке.

- **Поддержка чисел и массивов:**  
  Числа представляются как целые значения, массивы оформляются через `(list …)`.

- **Идентификаторы:**  
  Имена констант состоят только из заглавных латинских букв `[A-Z]+`.

- **Объявление констант:**  
  Константы задаются конструкцией `set ИМЯ = ЗНАЧЕНИЕ;`, где значение может быть числом, массивом или константным выражением.

- **Константные выражения:**  
  Вычисляются на этапе трансляции и могут включать:
  - сложение и вычитание (`+`, `-`)  
  - функции `pow()` и `abs()`  
  Константные выражения поддерживают вложенные ссылки на ранее объявленные константы.

- **Обработка ошибок:**  
  Программа выявляет лексические и синтаксические ошибки и сообщает о них через выходной файл.

---

## Тестовое покрытие

Все конструкции учебного конфигурационного языка, включая вложенные выражения и константные вычисления, покрыты тестами.  
Кроме того, показаны два примера конфигураций из разных предметных областей, что демонстрирует работу инструмента с разнообразными сценариями - сценарий сервера и сценарий физического эксперимента.
